===========
Tethr Portl
===========

Get started quickly
===================

::

    $ git clone git@github.com:tethr/portl.git
    $ cd portl/example
    $ virtualenv .
    $ cd ..
    $ example/bin/python setup.py dev
    $ cd example
    $ bin/pserve etc/portl.ini

Visit http://localhost:6543/

To reset wizard:

    $ rm var/wizard_status
